# This is a bicycle website    
___

This project is an online bike store that allows users to browse products, view detailed information, add items to the cart, and manage orders. The goal of the project is to provide a simple, fast, and modern shopping experience for bicycles and related accessories.

## Usage languages :
1. Javascvript
2. Python
3. HTML
4. CSS

## Usage Frameworks :

    - Next js
